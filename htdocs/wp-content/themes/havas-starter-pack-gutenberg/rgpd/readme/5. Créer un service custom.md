
Ajouter et bloquer un service custom
---------------------------------------------------------------

- copier le code ci-dessous en bas du fichier tarteaucitron.services.js
- remplacer les noms "servicecustom" par le nom du service à bloquer
- remplacer "type": "functional" par le type du service
	['ads', 'analytic', 'api', 'comment', 'social', 'support', 'video', 'functional']

- ajouter le lien vers les mentions légales concernant ce service dans "uri"
- ajouter les cookies à supprimer/autoriser dans le tableau "cookies"
- ajouter un script JS à charger avant que l'utilisateur donne son consentement dans "fallback" (facultatif)
- ajouter un script JS à charger au moment où l'utilisateur donne son consentement dans "js" (facultatif)

- vider le cache du navigateur ! IMPORTANT



Code à copier :
---------------

tarteaucitron.services.servicecustom = {
    "key": "servicecustom",
    "type": "functional",
    "name": "Service Custom",
    "uri": "http://perdu.com",
    "needConsent": true,
    "cookies": ['servicecustom_cookie_1', 'servicecustom_cookie_2', 'servicecustom_cookie_3'],	
    "js": function () {
        "use strict";		
		
    },	
    "fallback": function () {
        "use strict";
		
    }
};
