------------------------------------------------
DETECTER LE CHANGEMENT D'ETAT D'UN SERVICE
------------------------------------------------
Pour savoir si l'utilisateur a changé le consentement sur un service, i lexiste 2 évènements :

rgpd_consent_change :
- Se déclenche quand l'utilisateur change le consentement d'un service
- Se déclenche quand l'utilisateur change la valeur d'un bouton de consentement


rgpd_button_change :
- Se déclenche quand l'utilisateur change la valeur d'un bouton de consentement


Exemple :

document.addEventListener('rgpd_consent_change', function (event) {
	console.log(event);
});
