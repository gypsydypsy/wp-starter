* -------------------------------------------------------------------- *



------------------------------------------------
LE BANDEAU N'APPARAIT PAS :
------------------------------------------------

1/ 
********************************
Cela signifie qu'aucun service n'a été spécifié à tarteaucitron pour blocage.
Il faut rajouter un ou plusieurs services dans le footer du site.
Ex :
<script type="text/javascript">
    (tarteaucitron.job = tarteaucitron.job || []).push('youtube');
</script>


2/ 
********************************
Vérifier que les fichiers sont tous téléchargés dans la console Network
- tarteaucitron.js
- tarteaucitron.services.js




------------------------------------------------
UN SERVICE CUSTOM N'APPARAIT PAS DANS LA POPIN :
------------------------------------------------
- Vérifier la déclaration du service custom dans tarteaucitron.services.js
- Vider le cache de Chrome (pas uniquement avec Ctrl F5)
- Aller dans la console, dans settings, cocher Disable cache




------------------------------------------------
ERREUR JS : Cet objet ne gère pas cette action
------------------------------------------------
Se produit sur IE
Le custom event n'est pas supporté.
Solution : déplacer le polyfill CustomEvent tout en haut du fichier tarteaucitron




------------------------------------------------
ERREUR JS : Uncaught TypeError: Cannot read property 'title' of undefined
------------------------------------------------
Ouvrir le fichier de langue dans lang/
Vérifier que tous les types de services sont listés
Ex : 
    "functional": {
        "title": "Cookie fonctionnel",
        "details": "Nous vous recommandons de ne pas désactiver ce cookie."
    }
	
Vérifier que la propriété title existe pour chaque type de service
